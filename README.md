# WindoM

A Chrome extension that turns every new tab into a personal productivity dashboard — clock, weather, todos, calendar, Spotify, quick links, focus timer, and more. Built with a clean glassmorphism design.

## Run locally

There are two ways to run WindoM locally depending on what you need.

### Extension only (most features, no account required)

Works out of the box. Clock, weather, todos, backgrounds, quotes, quick links, focus timer — all fully functional without a backend.

See the [Local Setup Guide](https://yehudabriskman.github.io/WindoM/local-setup.html).

### Full stack (with Google Calendar + Spotify)

Requires running the backend locally with Docker. See the [Local Setup Guide](https://yehudabriskman.github.io/WindoM/local-setup.html).

---

## Tech stack

**Extension** — React 18, TypeScript, Vite, Tailwind CSS v4, CRXJS, Manifest V3

**Backend** — Fastify v5, Drizzle ORM, PostgreSQL 16, TypeScript ESM

---

## Privacy

WindoM does not track you. All personal data stays in your browser. An account is optional and only needed for Google Calendar and Spotify features. Read the full [Privacy Policy](https://yehudabriskman.github.io/WindoM/privacy.html).
